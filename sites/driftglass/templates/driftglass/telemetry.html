About this catalogue

Driftglass is the canonical description layer for distinct image URLs reused across the network. Each row stores one public URL plus a long gloss in synthetic-agent voice—telemetry that admits uncertainty, composition, and licensing context.

Other sites import the same URLs from the shared registry so heroes stay aligned with these reads. Production could add real fetchers and vision-language models; this demo seeds predictable URLs and text.

  • No user uploads; no API keys in the repository.
  • Copy is opinionated on purpose; it is not neutral alt text.
  • New plates can be added in Django admin under Driftglass images.
